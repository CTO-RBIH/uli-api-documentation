The Transliteration and Translation API provides a simple and effective means to transliterate or translate any text from standard English to selected Indian languages and vice versa. This service is offered by Bhashini and supports seamless language processing for financial applications and digital services.

This service is part of the Public Tech Platform for Frictionless Credit and enables multilingual support for enhanced customer experience across diverse linguistic communities in India.

## Key Features

- **Dual Language Processing**: Both transliteration (script conversion) and translation (meaning conversion)
- **Multi-Language Support**: Supports 6 major Indian languages plus English
- **Bidirectional Processing**: Convert from English to Indian languages and vice versa
- **Base64 Encoding**: Secure text processing with Base64 encoded inputs and outputs
- **Digital Signatures**: SHA256withECDSA algorithm for data integrity
- **Bhashini Integration**: Powered by India's national language technology platform

## Supported Languages

### Core Languages (Available in all versions)
- **English** (en) - Source and target language
- **Hindi** (hi) - India's most widely spoken language
- **Tamil** (ta) - Primary language of Tamil Nadu
- **Marathi** (mr) - Primary language of Maharashtra
- **Kannada** (kn) - Primary language of Karnataka

### Extended Languages (Available from v1.3 onwards)
- **Bengali** (bn) - Primary language of West Bengal
- **Telugu** (te) - Primary language of Andhra Pradesh and Telangana

## Service Information

- **API Version**: 1.3.1 (Latest)
- **Environment Support**: Sandbox and Production
- **Authentication**: JWT Token (6 hours validity in sandbox, 6 hours in production)
- **Request Method**: POST for all operations
- **Data Format**: JSON with Base64 encoded text
- **Service Provider**: Bhashini (Provider Code: 111)

## Technical Specifications

- **Input Format**: Base64 encoded text strings
- **Output Format**: Base64 encoded processed text
- **Character Support**: Full Unicode support for Indian languages
- **Security**: HMAC SHA-256 signatures for request integrity
- **Response Verification**: Digital signatures with public key validation

## Use Cases in Financial Services

- **Customer Communication**: Translate terms and conditions, loan documents
- **Data Entry**: Transliterate customer names and addresses
- **Compliance**: Provide regulatory information in local languages
- **User Interface**: Support multilingual banking interfaces
- **Document Processing**: Convert and verify multilingual documents
