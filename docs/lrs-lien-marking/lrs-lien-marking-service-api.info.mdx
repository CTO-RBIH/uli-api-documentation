The Lien Marking service sets mortgage details in the land records system upon successful loan issuance. The service provides digitally signed and tehsildar signed documents as proof of lien tagging the land in land record system. Supports multiple states including Madhya Pradesh, Tamil Nadu, Rajasthan, and Chhattisgarh.
