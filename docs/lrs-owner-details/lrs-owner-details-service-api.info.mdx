The Verify Land Ownership Details API service allows to verify the details of the owner of the land. It fetches the details from the land record system maintained by the state governments across 9 Indian states including Madhya Pradesh, Maharashtra, Uttar Pradesh, Tamil Nadu, Odisha, Andhra Pradesh, Tripura, Rajasthan, and Chhattisgarh.
