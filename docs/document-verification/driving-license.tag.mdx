The Driving License Verification API is a digital solution that allows businesses to verify an individual's driving license details online. It offers a paperless, real-time alternative to manual verification, enabling seamless onboarding and identity validation.

## Key Features

- **Digital Verification**: Paperless, real-time driving license verification
- **Comprehensive Data**: Retrieves detailed license and personal information
- **License Validation**: Verifies license authenticity and current status
- **Personal Details**: Access to holder's name, address, and license specifics
- **Real-time Processing**: Immediate verification results
- **Secure Processing**: Uses HMAC SHA-256 and JWT authentication

## Service Information

- **API Version**: 1.0
- **Environment Support**: Sandbox and Production
- **Authentication**: JWT Token based authentication
- **Response Format**: JSON with digital signature verification
